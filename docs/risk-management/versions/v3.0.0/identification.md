## مقدمه

## روش‌های شناسایی

### 1. طوفان فکری (Brainstorming)

جلسات گروهی برای شناسایی ریسک‌های احتمالی.

### 2. تکنیک دلفی (Delphi)

جمع‌آوری نظرات خبرگان به صورت ناشناس.

### 3. مصاحبه

مصاحبه با ذینفعان و خبرگان پروژه.

### 4. 🆕 تحلیل با هوش مصنوعی

استفاده از الگوریتم‌های یادگیری ماشین برای شناسایی خودکار ریسک‌ها.

```python
class AIRiskIdentifier:
    def identify_risks_from_text(self, project_description):
        """شناسایی خودکار ریسک‌ها از متن پروژه"""
        # استفاده از NLP برای استخراج ریسک‌ها
        risks = []
        keywords = ['تاخیر', 'کمبود', 'مشکل', 'خطر', 'نقص']
        
        for keyword in keywords:
            if keyword in project_description:
                risks.append(f"ریسک مربوط به {keyword}")
        
        return risks
```
