# راهنمای مدیریت پروژه

این سند راهنمای کامل مدیریت پروژه‌های PMO را ارائه می‌دهد.

## مقدمه

مدیریت پروژه یک فرآیند سیستماتیک برای برنامه‌ریزی، سازماندهی و کنترل منابع است.

## اهداف

- 📋 برنامه‌ریزی دقیق پروژه‌ها
- 👥 مدیریت تیم‌های پروژه
- 💰 کنترل بودجه و هزینه‌ها
- ⏱️ مدیریت زمان‌بندی

## فرآیندهای اصلی

### 1. شروع پروژه (Initiation)

در این مرحله:
- اهداف پروژه مشخص می‌شود
- ذینفعان شناسایی می‌شوند
- منشور پروژه تهیه می‌شود

### 2. برنامه‌ریزی (Planning)

برنامه‌ریزی شامل موارد زیر است:

- **تعریف محدوده (Scope)**
  - WBS (Work Breakdown Structure)
  - تعریف فعالیت‌ها
  - تخمین منابع

- **زمان‌بندی**
  - نمودار گانت
  - مسیر بحرانی (Critical Path)
  - تخصیص منابع

- **بودجه‌بندی**
  - تخمین هزینه‌ها
  - تخصیص بودجه
  - کنترل هزینه

### 3. اجرا (Execution)

مراحل اجرا:

1. تشکیل تیم پروژه
2. برگزاری جلسات کیک‌آف
3. انجام فعالیت‌های برنامه‌ریزی شده
4. تضمین کیفیت

### 4. پایش و کنترل (Monitoring)

ابزارهای پایش:

- گزارش‌های پیشرفت
- شاخص‌های کلیدی عملکرد (KPI)
- مدیریت ریسک
- مدیریت تغییرات

### 5. بسته‌شدن (Closure)

فعالیت‌های پایانی:

- تحویل نهایی
- مستندسازی
- ارزیابی عملکرد
- دروس آموخته

## نکات مهم

!!! warning "توجه"
    همیشه قبل از شروع پروژه، منشور پروژه را تهیه و تأیید کنید.

!!! tip "نکته"
    از ابزارهای مدیریت پروژه مانند Jira یا MS Project استفاده کنید.

!!! info "اطلاعات"
    این سند مطابق با استانداردهای PMBOK است.

## کد نمونه

برای محاسبه مسیر بحرانی:

```python
def calculate_critical_path(tasks):
    """محاسبه مسیر بحرانی پروژه"""
    critical_path = []
    for task in tasks:
        if task.float_time == 0:
            critical_path.append(task)
    return critical_path
```
